Partsfinders.co.za is an add on for Partfinders.co.za to used to request for vehicle parts wanted.
CAn also be reached www.partfinders.co.za/findapart
Database is from www.a-c-i.co.za

User request parts by completing form request
Emails is send to supplier to quote for parts wanted.
Suppliers click to quote, enter parts price and type (New/Used/After-market)
User receives email notification that quote or quotes is available.
User click link and select part wanted and enter Order Number
Platforms notify and transaction is concluded manually
